# CekRegoTerminal.AI

Retro-styled crypto analysis dashboard with mock AI signals.

## Run Locally
1. `npm install`
2. `npm run dev`

## Deploy
1. `npm run build`
2. Deploy `dist/` to Vercel/Netlify/GitHub Pages.

No API keys needed.
